Testing pomodoro timer API

Users

  • Note*: public profile = no credentials data
  • ../api/users : get all users public profiles (for leaderboard)
  • ../api/users/{id} : get user with that id (public profile)
  • ../api/users/login : pass in a user JSON through POST method (username and password is a must) to login and return the user credentials data and task list (2 JSON object, final JSON might be wrong)

    Example: {"id":"SE160947","email":"testing@test.com","password":"123456","timeFocusToday":13575,"timeFocusThisWeek":794850}

  • ../api/users/create : pass in new user JSON through POST method to create new user (there's a dublicate check and userid length must >= 8)

    Example: {"id":"SE160947","email":"testing@test.com","password":"123456","timeFocusToday":13575,"timeFocusThisWeek":794850}

    Note* : return status : 404 = ERROR ; 201 = SUCCESS ; 999 = DUBLICATE
  • ../api/users/{userId}/addTask : pass in new task JSON through POST method to add new task for that userId

    Example: {"id":0,"userId":"","name":"TestAddTask","duration":1209600,"startTime":1660755600000,"completeTime":1661965200000,"status":1}

    Note* : return status : 404 = ERROR ; 201 = SUCCESS ;

Tasks (Temporary Delete/Remove/Commented)

  • ../api/tasks : get all users
  • ../api/tasks/one : get one user
  • ../api/tasks/{userId}/{taskId} : get task with that id
  • ../api/tasks/ : pass in new task JSON through POST method to create new task

    Example: {"id":"17","userId":"SE160947","name":"OJT Signup","duration":7200,"startTime":1660323600000,"completeTime":1660410000000,"status":0}
